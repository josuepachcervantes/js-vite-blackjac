# BlackJack Vite

Pasos para ejecutar el proyecto:

1. Clonar repositorio
2. Ejecutar ```npm install``` para reconstruir los modulos de node
3. Correr el devServer ```npm run dev```
4. Abrirl el proyecto en: ```localhost:3000```
